How It Works

Three steps to optimal inventory

1

Enter Your Data

Upload your sales history, costs, lead times, and current stock levels. We accept CSV, JSON, or manual entry.

2

Developer API

Integrate OptiStock into your existing workflows

POST /v1/optimize Full optimization pipeline — demand forecast, simulation, order recommendation
Example Request 
curl -X POST https://api.optistock.com/v1/optimize \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key" \
  -d '{
    "items": [{
      "item_id": "SKU-001",
      "cost": 12.50,
      "sale_price": 29.99,
      "current_available": 150,
      "on_order_qty": 0,
      "lead_time_days": 14,
      "historical_data": [
        {"date": "2025-01-01", "quantity": 5, "available": 200},
        {"date": "2025-01-02", "quantity": 7, "available": 195}
      ]
    }],
    "tier": "premium",
    "cost_of_capital": 0.14
  }'
Example Response 
{
  "items": [{
    "item_id": "SKU-001",
    "optimal_outp": 175,
    "recommended_order_qty": 25,
    "expected_profit": 1247.50,
    "expected_daily_sales": 6.2,
    "expected_avg_inventory": 88.5,
    "demand_source": "ml_regression",
    "warnings": []
  }],
  "compute_time_ms": 1350
}
POST /v1/demand

Demand forecast only. Returns predicted ADS and variance without running the full simulation.

POST /v1/simulate

Direct simulation. Provide your own ADS and variance, skip demand forecasting entirely.

GET /health

Liveness check. Returns service status for monitoring and health checks.
